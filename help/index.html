Welcome to COVID-19 Application Help

About the Application

Covid-19 application is a standalone application which can be installed on a personal PC and Maintain the COVID-19 Data state-wise with the help of one or more administrators. Only Administrators can add new data of a new state or update the data of an existing state or can also delete the data of a state all-together.

New administors can be added only with the permission of an Exisitng Administrator (Other Admin's login credentials needed). How ever, any Admin acount can be deleted at any time, only by providing exiting login credentials. If an Admin forgot his password, he/she may try to re-create password through forgot password, or delete himself/herself from database and create another account with the help of other Admin's credetials.

All the data that is saved from the application, i.e., States' COVID-19 data and also User login credentials data, is stored in the file format using the pipeline concept. The data is retrived from the file when prompted by the application using the concept of split string in C++. The complete code of the application was programmed in Win32 C++ environment in Dev C++ IDE.

Using the COVID-19 Application

Admin Menu

Admin menu is the third menu in the menu bar of the application and it contains options for managing the administrator accounts (User accounts). The sub-menu options are,

  1. Register New User, for registering a new admin user who can access and edit the database.
  2. Forgot Password, for re-creating a new password for those who forgot their password.
  3. Delete Existing User, for deleting the existing admin user if he leaves the community.

Options Menu

Options menu is the second menu in the menu bar of the application and it contains options for managing the COVID-19 related data. The sub-menu options are,

  1. Add New Data, for adding the new states' information in to the application database.
  2. Update or Delete Exisiting Data, for updating or deleting the exisitng data of a state.

File Menu

File menu is the first menu in the menu bar of the application and it has the following options.

  1. Refresh Display, for refreshing the data displayed on the application after adding or updating database.
  2. Help, if you need any assistance regarding the working of this application, which brought you here! ofcourse.
  3. Exit, to exit the application.

About Menu

About menu is the fourth and last menu in the menu bar of the application and it contains general information as follows.

  1. WHO Guidelines on COVID-19, directs to the WHO oficial website where all the guidlines related to COVID-19 are documented.
  2. About COVID-19 Application, opens a window which shows application details like version, build, copyright details, etc.
